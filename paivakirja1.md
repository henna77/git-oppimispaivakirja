# Oppimispäiväkirja: Paikallinen git

__Mikä osion tehtävissä oli vaikeaa ja mikä helppoa? Mikä auttoi minua oppimaan? Miten selvitin esteet?__

Helppoa oli ymmärtää Gitin perusidea. Opin nopeasti käyttämään git status, koska kertoo missä tilassa olin. Aluksi kyllä kaikki tuntui vaikealta, mutta kun tarpeeksi etsin tietoa ja katsoin ohje videoita alkoi vähän helpottamaan.

## Osiossa käyttämäni Git-komennot

| Komento | Kuvaus |
| --------| ------ |
| git init | Luo uuden paikallisen git-repositorion |
| git status | Näyttää working directoryn ja staging area tilan |
